Tenho grande interesse em resolver problemas reais através de dados.

Atualmente estou desenvolvendo projetos envolvendo:

📊 Análise Exploratória de Dados (EDA)

📈 Visualização de Dados  
🗄 Modelagem de Banco de Dados
